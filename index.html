Mi az a MySQL?

A MySQL egy nyílt forráskódú relációs adatbázis-kezelő rendszer. Leggyakrabban webes alkalmazásokban használják, például a PHP és a WordPress mellett.

Főbb jellemzők:

  • Gyors és megbízható
  • Támogatja az SQL (Structured Query Language) használatát
  • Széles körben használják webfejlesztésben
  • Ingyenes (a közösségi verzió)

Egyszerű MySQL parancsok:

Adatbázis létrehozása:

CREATE DATABASE webshop;

Tábla létrehozása:


CREATE TABLE termekek (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nev VARCHAR(100),
    ar INT
);

Adat beszúrása:

INSERT INTO termekek (nev, ar) VALUES ('Laptop', 250000);

Adatok lekérdezése:

SELECT * FROM termekek;
